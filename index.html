🎵 Music App API

欢迎使用音乐应用 API！以下是可用的端点：

GET /api/health

健康检查端点，返回 API 状态信息

GET /api/songs

获取歌曲列表

POST /api/auth/register

用户注册端点

需要 JSON 数据：{ "username": "用户名", "phone": "手机号", "password": "密码" }

📖 使用说明

  • 所有 API 端点都支持 CORS
  • POST 请求需要发送 JSON 格式数据
  • API 响应格式为 JSON
